Help board:

  1. Press R to undo last node or edge created.
  2. Options that are RED are turned off.
  3. To connect nodes simply click on a node then the next.
  4. The base case will be a undirected edge, to create a directed edge click the option in the menu.
  5. "Autoconnect" will connect nodes that are close to each other.
  6. Bots will move with the edges, click "Hide Bots" to hide them.
  7. "Random Graph" will create a random graph based on input at the bottom.
  8. To find the shortest path between to nodes press the "Find Path" button.
  9. If you want to connect multiple nodes click the "Save Paths" button.
